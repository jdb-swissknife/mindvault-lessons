# MindVault Lessons

Static lesson property for MindVault individual-agent clients.

This repo intentionally ships the restored Agent Quest / lesson experience as plain static files so the lesson property cannot be accidentally overwritten by a separate landing-page build.

Routes:
- / -> Agent Quest overview
- /quest.html -> quest board
- /lesson1.html through /lesson11.html
- /lesson18.html
- /privacy.html

Custom domain:
- lessons.mindvaultstudio.net
